# Tech Poll
### Assets Used
- [Express 4.16.1][1]
- [Pug 2.0.0][2]
- [bodyParser 1.18.2][3]

[1]:https://expressjs.com
[2]:https://pugjs.org
[3]:https://github.com/expressjs/body-parser

### Install and set up a local dev environment
#### Prerequisites
- [Node JS][4]

[4]:https://nodejs.org

1. Download or clone the git repo
2. Run `npm install`
3. Run `node app` and navigate to `localhost:3000`
